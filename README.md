## Getting started

### Clone repo

```bash
# clone the repo
$ git clone https://github.com/radtket/sandstorm-nav-hover.git my-project

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

## Usage

```bash
# serve with hot reload at localhost:3000.
$ npm run start

# build for production with minification
$ npm run build
```
